# Vinharia Agnello - Sistema de Gerenciamento

### 🍷 Descrição do Projeto
Sistema básico para cadastro e análise de vinhos artesanais da Vinheira Agnello. Desenvolvido como parte do Checkpoint de JavaScript da FIAP.

## 🧠 Objetivo do Projeto
O objetivo deste sistema é aplicar os conceitos iniciais de JavaScript para gerenciar dados de vinhos artesanais, focando em:
*   Manipulação de variáveis e operadores.
*   Interação com o usuário via `prompt()` e `alert()`.
*   Exibição de dados organizados no `console.log()`.
